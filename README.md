## 详细说明   

### 分类
结果证明简单分类是完全错误的做法，他忽略了不同分数之间的联系，看成5个不同的类别导致完全欠拟合。   

### Logistic回归
训练模型回归一个潜变量 连续的 y* ，这个潜变量代表了模型对输入样本的预测分数。自定义 4 个切点，切点通过 scale 控制，也可以训练，但实验证明训练效果不佳。   
对潜变量和切点的差值应用Sigmoid函数，得到每个类别的累计概率。每个类别的概率不是独立的，而是与它之前的类别有关。   

$$
-\infty = c_{-1} < c_0 < c_1 < \dots < c_{K-1} < c_K = +\infty
$$


$$
P(Y \leq k \mid \mathbf{x}) = \sigma(c_k - \mathbf{x}\beta)
$$

$$
\begin{aligned}
P(Y = 0 \mid \mathbf{x}) &= \sigma(c_0 - \mathbf{x}\beta) \\
P(Y = k \mid \mathbf{x}) &= \sigma(c_k - \mathbf{x}\beta) - \sigma(c_{k-1} - \mathbf{x}\beta), \quad \text{对于 } k = 1, 2, ..., K-1 \\
P(Y = K \mid \mathbf{x}) &= 1 - \sigma(c_{K-1} - \mathbf{x}\beta)
\end{aligned}
$$

在计算得到每个类别的概率后，模型使用对数似然损失函数来衡量预测概率与真实标签之间的差距。   
模型使用交叉熵来评估预测类别概率与真实标签之间的差异。真实标签 y 会被用于选择正确的类别概率，并计算负对数似然损失。

### K-1分类
该模型使用4个分类头来学习类别之间的边界，并通过爬坡机制进行序数回归。每个分类头输出一个logit值，表示两个相邻类别之间的阈值，共有4个logits来区分5个类别。通过对这些logits应用Sigmoid函数，模型计算每个类别的概率，并基于这些概率预测最终的类别。

在爬坡机制中，模型通过学习每个类别的阈值，将标签转化为二进制形式，表示样本是否属于某个类别以上。模型使用二元交叉熵损失来计算logits与目标标签之间的误差，同时引入标准差正则化来稳定预测结果，避免过度偏向某些类别。这种设计使得模型能够有效地捕捉类别之间的顺序关系，并对具有顺序标签的任务进行优化。

### MSE 目前最好的方法
MSE 与 L1 方法解决此类问题是合理的，但是由于数据集由人类评价，每个人对不同词义的理由有较大差别，噪声较大，导致拟合效果也不是很好。
