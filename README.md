## 详细说明   
结果证明简单分类是完全错误的做法，他忽略了不同分数之间的联系，看成5个不同的类别导致完全欠拟合。   

MSE 与 L1 方法解决此类问题是合理的，但是由于数据集由人类评价，每个人对不同词义的理由有较大差别，噪声较大，导致拟合效果也不是很好。

有序分类 可能效果会更好，目前自定义损失函数还有些问题，还在施工中。
